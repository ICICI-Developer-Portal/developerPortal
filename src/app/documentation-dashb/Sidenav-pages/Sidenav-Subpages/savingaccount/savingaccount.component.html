Savings Account

A set of APIs to support Savings Account lifecycle, enabling innovative solutions to support end to end customer journey, from initiating the customer request for Savings Account, establishing customer relationship , enabling Savings Account process for creation and management through statements & services.

API Description
Create Saving Account This API will create the saving account for customer and in response will share the account number.
Block Card This api is used to block the debit or credit card temporary or permanently.
Unblock Card This api is used to unblock the debit or credit card.
Card Status This api is used to check the status of debit or credit card.
