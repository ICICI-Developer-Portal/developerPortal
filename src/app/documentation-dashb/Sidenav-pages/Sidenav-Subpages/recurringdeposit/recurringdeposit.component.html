Recurring Deposit

A set of APIs to support Recurring Deposit lifecycle, enabling innovative solutions to support end to end customer journey, from initiating the customer request for RD, establishing customer relationship , enabling RD process for creation and management through statements & services.

API Description
RD Creation This service will validate OTP entered by customer. On successful validation create RD using inputs provided by customer such as account no, amount, tenure and installment start date.
View RD Details This API is used to view RD numbers.
RD Closure This API is used to Closing RD Account.
