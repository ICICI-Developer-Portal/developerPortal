Pre approved offers

ICICI Bank prepares various product oriented offers for its existing customer in all the domains -Cards, Loans, Payments etc. These APIs allows you to directly access these offers and present it to your customers without any hassle. We provide you each step wise integration to present these offers to your customers and avail them without even visiting any branch, Relationship Manager or even our website.

API Description
Get Offer On Account This API is used get the offers of various ICICI Bank Products mapped to customer on the basis of account number
Get Offer On Customer ID This API is used for getting the offers various ICICI Bank Products mapped to customer on the basis of Customer ID.
Get Offer On Mobile This API is used for getting offers various ICICI Bank Products mapped to customer on the basis of mobile number.
Pre Approved Offers This API is used get the pre-approved mapped to customer on the basis of application number.
Block Offer This API is used to block the offers which are already availed by the customer.
