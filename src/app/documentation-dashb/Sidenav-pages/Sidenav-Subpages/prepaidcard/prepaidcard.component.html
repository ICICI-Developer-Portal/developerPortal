PayDirect Card- The PayDirect Card is a magnetic-strip-based reloadable Prepaid Card that addresses the concerns of an enterprise for payment of salaries, reimbursements, commissions and incentives to its employees/associates. With the API Suite of Pre-Paid Cards, the customers can easily create and manage all the prepaid cards.

API Description
Card Activation This API is used for card activation
Card To Card Transfer This API is used for transferring the amount between two cards.
Personal Info Enquiry This API is used to retrieve cardholders contact information which is available in CMS.
Account Statement This API is used for retrieving total Debit and Credit transaction details for particular months, Ledger Balance, Available Balance.
Update/Change PIN This API is used for changing or update the customer PIN.
Transaction Enquiry This API is used for getting details of any transactions.
