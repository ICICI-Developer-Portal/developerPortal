Getting Started

  • Sign up
  • Create an App
  • Select App
  • Test it Out

If you already have an account, then sign in and jump to step 2. Else follow the steps to create an account

# 1

Go to our sign-up page.

a. Enter your Credentials.

b. Verify your Mobile Number

c. Create Username and Password

# 2

Complete the registration by entering your details and create an account.

# 3

When authorized, an email will be sent to your registered email-id with a link to activate your account by logging in.

You now have access to your dashboard. From here you can create and configure your apps.

# 1

In your Dashboard, click Applications tab.

# 2

If there are no applications, you have to create an Application first to test the fuctionalities of APIs by adding them in the Application.

# 3

In the Application Information tab; you have to specify your application's name and on which platform your application is going to be running.

# 4

In API Management tab; you will choose the API(s) that you are going to be using in your application.

# 5

In the Auth tab; fields are optional.

# 6

After you click Save button, you will see your application in the Applications tab.
