CMS Payments

CMS Payments API enable the clients to upload payment file containing multiple transactions through NEFT; RTGS; IMPS; and FT. The client can also get transaction status through the API.

API Description
Instant Fund Transfer To transfer funds from one ICICI account to another ICICI account. Each request allows one transaction only.
Payment Process (File upload) To upload payment files containing multiple transactions. Web service request can contain multiple files in one request as well.
Reverse MIS To fetch the Reverse MIS from Bank to Customer
