Current Account

A set of APIs to support Current Account lifecycle, enabling innovative solutions to support end to end customer journey, from initiating the customer request for Current Account, establishing customer relationship , enabling Current Account process for creation and management through statements & services.

API Description
Create Current Account This API will create the current account for individual customer.
Mini Statement This api is used for checking the mini statement of customer.
Get Account Numbers This api is used to fetch the account numbers and userId mapped with the provided mobile number.
