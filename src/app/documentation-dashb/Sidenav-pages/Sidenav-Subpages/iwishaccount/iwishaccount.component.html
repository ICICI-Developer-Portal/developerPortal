iWish Account

A set of APIs to support IWish product lifecycle, enabling innovative solutions to support end to end customer journey, from initiating the customer request for iWish, establishing customer relationship , enabling iWish process for creation of goals and Goals management through funding and services.

API Description
Goal Creation This api is used to create merchant goal.
Editing Goals Details This api is used to edit merchant goal.
Adding Funds to a Goal This api is used to get transaction history.
Transaction History This api is used to get transaction history.
View Goal Service This api is used to view merchant goal.
Close Goal This api is used to close goal.
