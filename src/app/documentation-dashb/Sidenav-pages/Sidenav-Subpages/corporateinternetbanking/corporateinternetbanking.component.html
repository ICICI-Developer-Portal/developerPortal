Corporate Internet Banking

Corporate Internet Banking APIs enable the customer/partner to access ICICI Bank's internet portal services on his platform, like payment, bank statement, balance fetch and can apply for instant overdraft.

API Description
Registration TThis api is used for registration of company account with partner/aggregator.
Registration Status This API is used to check status of registration of company account with partner/aggregator.
Registration callback This api is used to send automatic update on successful registration of company account with partner/aggregator.
De-Registration This api is used for deregistering the account.
OTP Creation for Transaction This api is used to create OTP which is used for transaction.
Transaction This api is used to make payment with-in bank and outside bank (RTGS, NEFT,IMPS).
Transaction Status This api is used for transaction inquiry.
Transaction Callback This api gives transaction status.
Validate Registered Account This api is used for validating linked accounts.
Fetch Register Mobile No This api is used for fetching the mobile number linked to the account.
Account Balance Fetch This api is used for checking account balance.
Account Bank statement This api is used to get bank statement.
InstaOD offer check This api is to check for instant unsecured business loan(Overdraft) facility on the basis of account number.
Avail Insta OD Offer This api provide URL to avail instant overdraft facility offered by bank.
