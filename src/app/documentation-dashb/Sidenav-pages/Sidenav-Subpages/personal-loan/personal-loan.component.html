Personal Loan

Enabling innovative solution to enhance and ease the customer journey for personal loans through a set of APIs supporting the end to end processes : avaialble offers, request for loan , loan processing and post loan disbursement services. The key available APIs are:

API Description
Generate OTP This API is leveraged to create and deliver OTP as SMS for authentication or confirmation as required in the process.
Get Offer This api is used to provides the details of the loan offer that is available for the customer based on the bank's evaluation process. For PL, the api shows the available PL offer to the customer.
Eligibility Check This api is used to enables the customer to change the offered loan amount or tenure based on the requirement and submit the changed loan offer for evaluation.
Bank Statement Upload This api is leveraged to upload bank statements of customer to bank for verification process which is used for underwriting.
Document Upload This api is uesd to leveraged to upload documents required as per the process.
Disbursement This API is leverged to initiate the loan disbursement process. This API initiates the process of disbursemnet , booking , updation and successful / failure update to the customer.
