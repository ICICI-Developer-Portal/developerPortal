Paylater

PayLater is a digital credit product, similar to a Credit Card. With it you can make your purchases today and make payments within 45 days of the transaction date and no interest is charged. The API integration allows you to facilitate your customers with the PayLater product on your platform.

API Description
Account Creation This API will be used to check the status of pay-later account by mobile number or transaction id.
Paylater Balance Inquiry This API will be used for balance inquiry in ICICI PayLater A/c.
Bill Inquiry This API will be used for bill inquiry in ICICI PayLater A/c.
Pay Dues This API will be used to make Pay Dues of ICICI PayLater A/c.
OD A/c Discovery This API will be used for OD account discovery basis on the mobile number.
