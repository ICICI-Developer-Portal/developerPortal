IMPS API

This API is used to make IMPS payment process.

API Description
P2P - Person to Person Fund Transfer Enables funds transfer from remitter BC account to beneficiary account using MMID and MobileNumber.
P2A - Person to Account Fund Transfer Enables funds transfer from remitter BC account to beneficiary account using IFSC and Account Number details of Beneficiary.
IMPS MMID Inquiry This is inquiry transaction to know the status for MMID generation.
IMPS Transaction Inquiry This API will return the status of transaction specific to given Reference Retrieval Number.
