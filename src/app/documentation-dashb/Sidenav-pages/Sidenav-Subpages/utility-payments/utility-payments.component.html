Utility Payments

BBPS is the centralized system facilitating all bill collection needs. The interoperable platform allows the customer of one bank to pay bills of a biller acquired by another bank. The Utility payments API suite allows you to connect directly with the BBPS network of India. The product not only enables collection of bills through online Payment Gateway based platforms but also through an extensive network of physical merchants and agents.

API Description
Get Paid Bill This api is used to fetch total number of bills paid by the agent.
Get Biller List This api is used to fetch a biller from a selected category.
Validate Payment Amount This api is used to validate payable amount of a biller.
Pay Bill This api is used to payment for pay only amount.
Validate Customer Parameters This api is used to determine whether the customer parameters entered by user are correct.
Get Tax Amount This api is used to calculate all kind of taxes.
