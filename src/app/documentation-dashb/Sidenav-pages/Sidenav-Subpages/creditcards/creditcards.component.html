Credit Cards

Credit Cards API support the end to end customer journey, from initiating the customer request for credit cards, available cards based on customer profile , enabling request processing , credit card creation and services to manage credit cards. The key available APIs are:

API Description
Credit Cards Get Offer This Api is leveraged to Avail all the Credit Card offers available to the customer on the basis of Linked Mobile Number after OTP verification.
Card Details This Api is leveraged to display all the detalis linked to the credit card depending upon the offer available to the customer.
Card Creation This Api is leveraged to create the card as the final acceptance step.
Block Card This api is used to block the debit or credit card temporary or permanently.
Unblock Card This api is used to unblock the debit or credit card.
Card Status This api is used to check the status of debit or credit card.
