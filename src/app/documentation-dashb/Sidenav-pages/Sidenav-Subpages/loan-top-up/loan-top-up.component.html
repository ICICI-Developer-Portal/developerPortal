Loan Top-Up

Enabling innovative solution to enhance and ease the customer journey for top-up loans for mortagage and automobile loans, through a set of APIs supporting the end to end processes : avaialble offers, request for loan , loan processing and post loan disbursement services. The key available APIs are:

API Description
Offer Fetch This api is used to display the offer details with EMI & interest rate.
Generate Agreement This api is used to provides the details of the loan offer that is available for the customer based on the bank's evaluation process. For PL, the api shows the available PL offer to the customer.
Disbursement This api is used to for loan disbursement.
