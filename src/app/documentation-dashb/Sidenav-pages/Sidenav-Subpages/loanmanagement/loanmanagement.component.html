Loan Management

A set of APIs that support loans management for customers through statements , transaction details , payment schedules , payments , closure and other services. The key available APIs are:

API Description
Get Loan Account Details This API is used to get the loan details on the Account.
Repayment Details This API is used for Repayment Details of Loan.
Profile Information This API is used to get Profile Information of Loan Account Holder.
Disbursement Details This API is used to know the Disbursement Details of the given Loan account holder.
Loan Credential Verification Check This API is used for Loan credential verification against Loan Account Number.
Get IT Certificate This API will be used to get IT certificate based on given Loan.
