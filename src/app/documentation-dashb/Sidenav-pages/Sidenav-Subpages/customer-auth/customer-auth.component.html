Customer Authentication

Leverage these APIs for customer authentication based on mobile number or customer account etc.

API Description
Generate OTP This API will be used to create the OTP and the OTP will be send to customer Mobile Number.
OTP Verification This API will be used to verify the OTP.
Aadhar OTP generation This API is used to generate OTP for Aadhar
Aadhar OTP verification This API is used for OTP verification for Aadhar.
